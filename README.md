# 🚗 Análise do Mercado Automotivo no Brasil

## 🔗 Acesse o projeto
Notebook completo disponível neste repositório.

## 📊 Objetivo

Este projeto tem como objetivo analisar o crescimento das vendas de veículos no Brasil ao longo dos anos, com foco na evolução dos veículos elétricos e híbridos em comparação aos veículos a combustão.

---

## 📂 Fonte de Dados

Os dados utilizados neste projeto representam vendas de veículos segmentadas por:

* Ano
* Tipo de veículo (Combustão, Híbrido, Elétrico)
* Volume de vendas

---

## 🧹 Tratamento dos Dados

* Importação dos dados utilizando Python (pandas)
* Verificação de tipos de dados
* Análise exploratória inicial

---

## 📈 Análises Realizadas

### ✔️ Evolução das vendas ao longo do tempo

* Identificação de tendência de crescimento
* Queda observada em 2020, possivelmente associada à pandemia
* Recuperação consistente nos anos seguintes

### ✔️ Comparação por tipo de veículo

* Veículos a combustão dominam o mercado
* Veículos híbridos apresentam crescimento relevante
* Veículos elétricos ainda possuem menor volume

### ✔️ Participação de mercado

* Veículos elétricos representam aproximadamente **5,37%** do total de vendas

### ✔️ Crescimento anual

* O maior crescimento foi observado em **2022**, indicando recuperação do setor

---

## 💡 Principais Insights

* O mercado automotivo brasileiro apresenta tendência de crescimento no período analisado
* A queda em 2020 foi seguida por recuperação significativa
* Veículos híbridos atuam como uma transição tecnológica no mercado
* Veículos elétricos, embora ainda com baixa participação, demonstram potencial de crescimento
* Veículos a combustão permanecem dominantes, com crescimento mais estável

---

## 🛠️ Ferramentas Utilizadas

---

## 📌 Conclusão

O estudo indica que o mercado automotivo brasileiro está em processo de transformação, com crescimento gradual de tecnologias mais sustentáveis, embora ainda em estágio inicial. O cenário aponta oportunidades futuras para expansão dos veículos elétricos.
